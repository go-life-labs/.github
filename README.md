# GoLife Labs

**GoLife Labs is building a personal AI life mentor for better habits, healthier routines, and more intentional living.**

GoLife started as a simple CLI tool and is growing into a personal life operating system: a place where health data, nutrition, habits, journaling, goals, and AI-powered reflection come together.

Our mission is simple:

> Help people understand themselves better and make better decisions every day.

## What is GoLife?

GoLife is a privacy-conscious, modular life tracking and coaching system.

It connects data from tools like wearables, nutrition trackers, calendars, habit systems, and personal notes — then turns that data into useful insights through AI.

The goal is not just to collect data.

The goal is to create clarity, direction, and progress.

## Core Ideas

* **Personal-first** — built for individual growth, not mass surveillance.
* **Privacy-conscious** — your life data should stay under your control.
* **Modular by design** — integrations can be added step by step.
* **AI-assisted, not AI-controlled** — the system should guide, not decide for you.
* **Small daily progress** — better living comes from consistent reflection and action.

## Planned Components

* `go-life-cli` — command-line interface for daily tracking and interaction
* `go-life-api` — backend API for data access and integrations
* `go-life-worker` — background jobs for syncs, summaries, and analysis
* `go-life-infra` — Terraform-based cloud infrastructure
* `go-life-integrations` — connectors for services like Oura, Yazio, and more
* `goli` — the personal AI mentor inside GoLife

## Philosophy

GoLife is not meant to optimize every second of your life.

It is meant to help you notice patterns, understand trade-offs, and move toward the person you want to become.

Less noise.
More clarity.
Better next steps.

## Status

This project is in an early experimental stage.

The first version is focused on one user, one life, and one simple question:

> What can I do today to live a little better tomorrow?
